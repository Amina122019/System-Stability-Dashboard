# System-Stability-Dashboard
An interactive Power BI dashboard designed to monitor system execution performance across two interfaces: Exposed and External. The dashboard provides a comprehensive view of execution status, enabling users to track success and failure rates, identify failed processes, and monitor overall system stability.

Key Features
📊 KPI Cards displaying:

Total Executions
Successful Executions
Failed Executions
Last Successful Execution Time
Last Failure Time


🔄 Performance comparison between the Exposed and External interfaces, including success and failure rates.
🍩 Overall system success and failure rate visualization.
📋 Detailed process execution table with failed processes highlighted in red for quick identification and troubleshooting.
🔎 Interactive filters for interface, date, and time to support focused analysis.

Purpose
This dashboard enables operations and support teams to monitor system health across both interfaces, quickly identify failed executions, analyze performance trends, and investigate affected processes to improve system reliability.

Tools Used: Power BI, Power Query, DAX
